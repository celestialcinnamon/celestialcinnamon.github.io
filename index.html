Francis Rubio

I am a Web developer and designer based in the Philippines. With 3 years of freelance development experience in my arsenal, I have worked on desktop, mobile, and Web applications.

Currently, I work on Antares Programming, an initiative to provide high-quality Web development resources in Filipino, the national language of the Philippines.

Notable Projects

UI Designs

A collection of UI concepts implemented from sample designs I have either come across around the Web, or have made myself.

Quantarks

An Android app that utilizes augmented reality to show information about the elements of the periodic table in interesting ways. It is built with Unity and Vuforia.

Antares Blog

A blog that I am maintaining where I talk about the different technologies now available on the Web. My primary target audience are Filipino budding Web developers who, although already capable, still needs to know more about the lesser-known features of the Web.
