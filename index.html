Quantarks

Quantarks is an augmented reality app we are currently working on for our thesis project. It is an app that renders 3D atomic models of each element of the periodic table.

It features voice overs that tells the story of how each element was discovered, how it is being used, and other fun facts about it.

It is an Android app built with Unity Engine and C#.
